# ClaudePilot for OpenClaw

**The AI-guided way to get started with OpenClaw.**

ClaudePilot is a single prompt you paste into [Claude](https://claude.ai) (free tier works). Claude becomes your personal setup consultant — it asks what you want to build, figures out what you need, and walks you through every step from zero to running bots with memory and smart model routing.

## What You Get

- A personal AI assistant that asks the right questions first
- Hardware recommendations matched to your budget and goals
- Step-by-step setup of [OpenClaw](https://github.com/GuyMannDude/openclaw) agents
- Memory system setup with [Mnemo Cortex](https://github.com/GuyMannDude/mnemo-cortex-v2)
- Cost optimization with [Sparks Router](https://github.com/GuyMannDude/sparks-router)
- Multi-bot configurations if you need more than one agent

## How to Use

1. Open [claude.ai](https://claude.ai) (free account works)
2. Start a new conversation
3. Copy the entire contents of [CLAUDEPILOT-OPENCLAW.md](CLAUDEPILOT-OPENCLAW.md)
4. Paste it into Claude
5. Follow the conversation — Claude will guide you through everything

That's it. No terminal commands to memorize, no docs to read first. Just a conversation.
