# ClaudePilot for OpenClaw

**The AI-guided way to deploy OpenClaw.** No developer required.

Pick your tier. Copy the guide. Paste it into Claude. That's it.

---

## Standard — Free

**Works with any Claude account, including free.**

Claude becomes your installation coach. It asks about your setup, explains every decision, and walks you through commands one at a time. You learn while you build. Works with claude.ai, ChatGPT, Gemini, or any capable LLM.

**Guide:** [CLAUDEPILOT-STANDARD.md](CLAUDEPILOT-STANDARD.md)

**How to use:**
1. Open [claude.ai](https://claude.ai) — free tier works perfectly
2. Start a new conversation
3. Copy everything below the instruction line in the guide
4. Paste it as your first message
5. Claude guides you through everything

---

## Pro — Max Subscription

**Claude Opus 4.6 becomes your deployment architect.**

Opus installs its own build tools (Claude Code), maps your infrastructure, designs the architecture, builds everything, and hardens security. Tailored to your business. You answer questions about what you want — Opus does the rest.

**Guide:** [CLAUDEPILOT-PRO.md](CLAUDEPILOT-PRO.md)

**What you need:**
- [Claude Max subscription](https://claude.ai) ($100/month) — gives you unlimited Opus 4.6
- A machine to deploy on (Linux recommended, macOS works, Windows via WSL)
- That's it. Opus handles the rest.

**How to use:**
1. Open [claude.ai](https://claude.ai) — make sure you're on Opus 4.6
2. Start a new conversation
3. Copy everything below the instruction line in the guide
4. Paste it into Claude
5. Opus takes over from there

## What Gets Built (Both Tiers)

- **OpenClaw** — the agent framework running your bot(s)
- **Channel connections** — Telegram, Discord, WhatsApp, or web chat
- **Mnemo Cortex v2** — persistent memory so your bot remembers across conversations
- **Sparks Router** — smart model routing that cuts costs without cutting quality
- **Systemd services** — everything starts on boot, runs 24/7
- **Backups and monitoring** — daily snapshots, alerts if something breaks
